## Файлы изменены:

2. `index.html` - добавлен аккордеон и исправления видео
3. Подключен новый CSS файл в `<head>`

## Как применить:

1. Скопируй файл `mobile-fixes.css` в папку `src/styles/`
2. Убедись, что в `index.html` подключен файл:
   ```html
   <link rel="stylesheet" href="src/styles/mobile-fixes.css" />
   ```
3. Загрузи все измененные файлы в GitHub
4. Сайт обновится автоматически через 1-2 минуты
