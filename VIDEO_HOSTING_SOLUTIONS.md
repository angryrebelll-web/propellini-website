# Решения для хостинга больших видео файлов

## Проблема
GitHub имеет ограничения:
- Максимальный размер файла: 100 MB
- Рекомендуемый размер: до 50 MB
- Для файлов > 50 MB нужен Git LFS (Git Large File Storage)

## Варианты решения

### ✅ ВАРИАНТ 1: Git LFS (Рекомендуется для GitHub)

**Установка Git LFS:**
```powershell
# Установите Git LFS: https://git-lfs.github.com/
# После установки выполните:
git lfs install
git lfs track "*.mp4"
git add .gitattributes
git add public/images/video/*.mp4
git commit -m "Add video files with LFS"
```

**Преимущества:**
- Видео хранится на GitHub через LFS
- Бесплатно до 1 GB хранилища
- Видео доступно по прямой ссылке

**Недостатки:**
- Нужна установка Git LFS
- Загрузка может быть медленной

### ✅ ВАРИАНТ 2: Cloudflare Stream (БЕСПЛАТНО и ЛУЧШЕЕ решение)

**Что это:**
- Бесплатный хостинг видео от Cloudflare
- 100,000 минут видео в месяц бесплатно
- Автоматическая оптимизация для разных устройств
- CDN по всему миру

**Как использовать:**
1. Зарегистрируйтесь на https://dash.cloudflare.com/sign-up
2. Перейдите в Stream
3. Загрузите видео файлы
4. Получите embed код или прямую ссылку
5. Вставьте в HTML

**Преимущества:**
- Полностью бесплатно
- Быстрая загрузка
- Автоматическая оптимизация
- Поддержка разных разрешений

### ✅ ВАРИАНТ 3: YouTube (БЕСПЛАТНО)

2. Получите embed код

**Недостатки:**

### ✅ ВАРИАНТ 5: Яндекс.Диск / Google Drive

**Как использовать:**
1. Загрузите видео на Яндекс.Диск или Google Drive
2. Сделайте файл доступным по ссылке
3. Получите прямую ссылку на скачивание
4. Используйте эту ссылку в HTML

**Получение прямой ссылки:**
- **Яндекс.Диск**: https://disk.yandex.ru → Поделиться → Получить ссылку
- **Google Drive**: Правая кнопка → Поделиться → Копировать ссылку

**Преимущества:**
- Бесплатно
- Удобно
- Большое хранилище

### ✅ ВАРИАНТ 6: Оптимизация видео (Уменьшение размера)

**Инструменты для сжатия:**
1. **HandBrake** (бесплатно): https://handbrake.fr/
2. **FFmpeg** (командная строка)
3. Онлайн сервисы: https://www.freeconvert.com/video-compressor

**Рекомендуемые настройки:**
- Разрешение: 1920x1080 (Full HD) или меньше
- Битрейт: 2-5 Mbps для хорошего качества
- Формат: MP4 (H.264)
- Целевой размер: 20-30 MB на минуту видео

**Команда FFmpeg для сжатия:**
```bash
ffmpeg -i input.mp4 -vcodec h264 -acodec mp2 -crf 23 -preset medium output.mp4
```

## РЕКОМЕНДАЦИЯ

**Лучший вариант для вашего проекта: Cloudflare Stream**

Почему:
1. ✅ Полностью бесплатно
2. ✅ Профессиональное качество
3. ✅ Быстрая загрузка
4. ✅ Автоматическая оптимизация для мобильных
5. ✅ Можно встроить в `<video>` тег или использовать iframe

**Как это сделать:**
1. Зарегистрируйтесь на Cloudflare
2. Загрузите видео
3. Получите прямую ссылку или embed код
4. Замените в `index.html`:

```html
<!-- Вместо: -->
<source src="./public/images/video/teaser-desktop.mp4" type="video/mp4" />

<!-- Используйте: -->
<source src="https://customer-XXXX.cloudflarestream.com/VIDEO_ID/manifest/video.m3u8" type="application/x-mpegURL" />
```

Или используйте embed код от Cloudflare.

## Если хотите оставить видео локально

Для локального использования (не для GitHub):
- Просто оставьте видео в папке `public/images/video/`
- Используйте локальный сервер (python -m http.server)
- Для production - используйте веб-хостинг с поддержкой больших файлов
