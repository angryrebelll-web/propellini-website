# propellini-website

Propellini - сайт по оклейке автомобилей защитной пленкой.

## Структура проекта

- `index.html`: Главный HTML файл
- `src/styles/main.css`: Все стили сайта
- `script.js`: JavaScript для интерактивных элементов
- `public/`: Медиа файлы (изображения, видео, шрифты, логотипы)

## Технологии

- HTML5
- CSS3
- JavaScript
- GSAP (анимации)

## Установка и запуск

1. Откройте `index.html` в браузере
2. Или запустите локальный сервер:
   ```bash
   python -m http.server 8000
   ```
   Затем откройте http://localhost:8000
